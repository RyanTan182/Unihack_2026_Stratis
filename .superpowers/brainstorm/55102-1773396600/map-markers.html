Interaction Summary

Tree loaded, nothing selected Purple dots on highest-concentration country per leaf node. No labels. Size = concentration %.
Node clicked in tree All other dots fade out. Selected node's countries appear with colored dots + percentage labels. Colors match the concentration bar in the detail panel.
Click away / deselect Returns to default state (all leaf nodes, single dots).
